# Alert

This is an Alert component.

```jsx

// /**
//  * debug: true
//  */

import { CuAlert } from 'colorui-react';

export default () => <CuAlert />
```

## API

| 属性 | 说明     | 类型                                         | 默认值 |
| ---- | -------- | -------------------------------------------- | ------ |
| kind | 警告类型 | 'info'/'positive'/'negative'/'warning'非必填 | 'info' |
